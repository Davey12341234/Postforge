# CLAUDE.md - bbGPT Project Context

@AGENTS.md

## Project
- Name: bbGPT.ai
- Framework: Next.js App Router
- Primary LLM path: Z.AI / GLM (`z-ai-web-dev-sdk`)
- Fallback LLM: OpenAI
- Image + attachment stack: Gemini server routes
- Billing: Stripe
- Email: Resend
- Persistence: Postgres via Drizzle

## Env Behavior That Matters
- `ANTHROPIC_API_KEY` is not used by active app code in `src/` (only historical `_archive/` files).
- Auth feature flags are `"1"` based booleans in code:
  - `BBGPT_USER_AUTH=1`
  - `NEXT_PUBLIC_BBGPT_USER_AUTH=1`
- Stripe toggles are also `"1"` style:
  - `STRIPE_CHECKOUT_AUTO_TAX=1`
  - `STRIPE_GRACE_PAST_DUE=1`
- `DATABASE_URL` should be a Postgres URL for this repo's live persistence path.
